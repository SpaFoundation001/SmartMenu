# 🍽️ Smart Menu

A modern digital restaurant menu website with a luxury UI.

## 🚀 Features
- Beautiful food cards UI
- Category filtering (Cake, Drinks, Snacks, etc.)
- Search + Voice search 🎤
- Cart system 🛒
- Product view page
- 3D model support (.glb files)
